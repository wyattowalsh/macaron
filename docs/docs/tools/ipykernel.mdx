[`ipykernel`](https://ipykernel.readthedocs.io/) is the reference Jupyter kernel built on top of IPython, providing a powerful environment for interactive computing in Python. It enables Jupyter applications to execute Python code, offering features like introspection, rich media, shell syntax, tab completion, and history.
