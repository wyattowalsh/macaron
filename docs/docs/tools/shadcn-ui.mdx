## Usage Example

To use a component from `shadcn/ui`, follow these steps:

1. **Initialize your project:**

   Use the CLI to set up your project and install necessary dependencies:

   ```bash
   npx shadcn@latest init
   ```

   This command will prompt you to configure your project settings. :contentReference[oaicite:1]{index=1}

2. **Add a component:**

   Add the desired component to your project using the `add` command. For example, to add a button component:

   ```bash
   npx shadcn@latest add button
   ```

   This will copy the button component code into your project, allowing you to customize it as needed. :contentReference[oaicite:2]{index=2}

3. **Use the component in your application:**

   Import and use the component in your React application:

   ```javascript
   import { Button } from '@/components/ui/button';

   function App() {
     return <Button>Click Me</Button>;
   }

   export default App;
   ```

   This example demonstrates how to integrate the button component into your application. :contentReference[oaicite:3]{index=3}
