[`Docusaurus`](https://docusaurus.io/) is an open-source static site generator designed to build documentation websites efficiently. It leverages modern web technologies, including React, to create fast, customizable, and user-friendly documentation sites. Docusaurus offers features such as versioning, internationalization, and a plugin system, making it a versatile tool for managing and presenting project documentation.

---

## Usage

First, navigate to the `docs` directory:

```zsh
cd docs
```

Then, to run a dev server:

```zsh
npm run start
```
